# Smart Motorcycle Crash Detection and Alert System

This project implements a real-time motorcycle accident detection system
using an embedded device (Raspberry Pi Pico W), cloud services, and an
Android mobile application.

## Features
- Automatic crash detection using MPU6050
- GPS location tracking (NEO-6M)
- Cloud alerts via Firebase & IFTTT
- Android app for real-time monitoring
- Email and notification alerts

## Project Structure
- Embedded_System/: MicroPython code for Pico W
- Android_Application/: Android app source code
